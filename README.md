# 📊 Exploratory Data Analysis (EDA) on Retail Sales Data

## 📌 Project Overview

This project is completed as part of the **Oasis Infobyte Data Analytics Internship**.

The objective of this project is to perform **Exploratory Data Analysis (EDA)** on a retail sales dataset to discover customer purchasing patterns, sales trends, product performance, and generate actionable business insights.

---

## 🎯 Objectives

- Perform data cleaning and preprocessing
- Analyze sales trends over time
- Explore customer demographics
- Identify top-selling products and categories
- Visualize relationships between variables
- Provide business recommendations based on findings

---

## 🛠️ Tools & Technologies

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

---


## 📈 Project Workflow

### 1. Data Loading

- Imported dataset using Pandas
- Checked dataset dimensions

### 2. Data Inspection

- Data types
- Missing values

### 3. Data Cleaning

- Dataset is cleaned already no need to clean 

### 4. Exploratory Data Analysis

Performed the following analyses:

- Monthly Sales Trend
- Quarterly Sales Trend
- Customer Age Distribution
- Gender Distribution
- Top 10 Best Selling Products
- Revenue by Product Category
- Correlation Heatmap
- Revenue by Payment Method

### 5. Business Insights

Generated meaningful insights from the visualizations and data analysis.

## 💡 Key Insights

Some of the insights obtained include:

## 🚀 Business Recommendations

- Increase inventory during peak sales periods.
- Focus marketing on high-performing product categories.
- Offer personalized promotions based on customer demographics.
- Improve sales of low-performing products using discounts or bundles.
